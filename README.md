The project I am the proudest of would be "Ballroom Madness" that we submitted for the Global Game Jam 2026, for it I was the project managed and it was a very satisfying challenge as 3/5 of the members of the team had never previously worked collaboratively with others.

In the end we successfully delivered a working game within the 7 days that we had, despite all the challenges. Not many devs or teams can say they got a working game during their first game jam!
